# Principais comandos
`git add .` Adiciona as alterações de cód. no proximo commit 
`git commit`
`git status`
`git push` Empurra os commits locais para o origin
`git commit -m "minha mensagem aqui"`
`git pull` Puxa da origin para local
`git checkout develop`
`git branch -d master`
